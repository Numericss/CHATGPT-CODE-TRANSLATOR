# AI Code Translator

Use AI to translate code from one language to another.

![AI Code Translator](./public/demo.png)

## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/Numericss/CHATGPT-CODE-TRANSLATOR.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
